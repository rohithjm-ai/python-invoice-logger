# Invoice Calculator with Logging (Python)

This project is a simple **Invoice Calculator** built using Python.  
It demonstrates how to use the **logging module** to track events such as:
- Start and end of calculations
- Items being added
- Handling negative/incorrect prices
- Warning messages

This project is useful for beginners learning:
- Python basics  
- Logging (INFO, WARNING, DEBUG)
- Clean code structure  
- File logging using `logging.FileHandler`
